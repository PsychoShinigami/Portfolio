Yea so I wanted to learn HTML and more which is required to make webs. So i decided to make a project while learning.
So I made this a simple browser-based game where players rate different outfits and decide the winner based on votes.
How it works:
Enter the number of players.
Each player enters their name.
Players take turns rating each outfit (like or dislike).
Each outfit gets a score based on votes.
After all players vote, the outfit with the highest score wins.
Notes:
Each player can vote only once per outfit.
Scores can go negative if disliked.
In case of equal scores, it shows a tie.
How to run:
Download or clone the project.
Open the HTML file in your browser.
Enter number of players and start playing.
