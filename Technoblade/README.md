Technoblade never dies
A css focused website tribute to the minecraft legend, technoblade. Built for Technoblade fans and to submit in hackclub(falvor-town).
Features:
  Sided hover menus for navigation and video archives.
  Custom sections for timeline, charity work, and legendary wins.
  Centered respect bar with a glow effect.
  Custom css gradients, box-shadow lighting.
Tech stack:
  HTML: Semantic structure for project modules.
  CSS: Mainly things done by CSS, as I built this project to learn more about CSS.
  Javascript: Logic for output of the buttons.
How to use:
  Open Demo in flavor-town web.
  Hover over things to see menu.
